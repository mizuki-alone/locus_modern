# Locus — 未実装機能リスト（旧版にあり新版に未実装）

## ノード操作
- 複製（子孫含む）: ノードとその子孫をまるごとコピー
- これだけ複製: ノード単体のみコピー（子孫なし）
- 空データ挿入: 空のノードを挿入

## 表示モード
- 印刷モード: サブツリーを読み取り専用で別ウィンドウ表示
- HTML展開: 参照を再帰展開してHTML出力
- TEXT展開: 参照を再帰展開してプレーンテキスト出力

## 連番リスト（OL）
- 連番の表示（1. 2. 3. ...）
- OLバッジ表示（親ノードに「OL」マーク）

## 参照・リンク機能
- ID追加: ノードにタイムスタンプベースの一意IDを付与
- ID削除: ノードからIDを除去
- 参照（ref）: 他のエンティティへの参照リンク `!{ref:EntityName}`
- 参照先: 参照リンクの作成
- 参照の再帰展開: HTML/TEXT/MD展開時に参照を実体に置換
- 子プレースホルダ: `!{child:EntityName:childNum}` による部分参照

## ファイル生成
- ファイル生成: `!{file:path}` メタデータを持つノードからファイルを生成
- 下層生成: サブツリー内の全 `!{file:path}` ノードから一括ファイル生成
- 確認: `!{file:path}` のURLを開いてプレビュー

## 認証・セキュリティ
- パスワード認証（マスターキー）
- パスワード変更機能
- 初回セットアップ（サーバー環境自動検出）
- Cookie セッション管理
- 二重暗号化（CryptString）

## UI パネル
- Methods パネル（左固定）: 各種操作ボタン一覧
- ObjectsⅠ パネル: 先頭26エンティティのドラッグ可能リスト
- ObjectsⅡ パネル: 残りエンティティ
- 上部コントロールバー: 検索・再読込等

## D&D 操作（メソッドボタン → ノード）
- 印刷・HTML展開・TEXT展開・MD展開・確認: ボタンをノードにドロップで実行
- 削除・複製: ボタンをノードにドロップで実行
- URLを開く: ノード内URLを新しいウィンドウで開く

## その他
- スクロール位置の保存・復元（操作後に元位置に戻る）
- ナビゲーションバー（モード切替・ログアウト）
- 二重送信防止（postAction ロック）
- テーマカラー設定（setup.ini の bgColorListMain）
